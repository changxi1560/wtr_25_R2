### [2.2.1] - 2025-3-14
- 通过上位机（电脑的串口助手）更改L1测距传感器，更改为上电后的激光关闭
- 试图使用环形缓冲区进行数据接收以提高响应速率，但不好使，无法正常数据解析，故使用会原来的单缓冲区实时接收数据
- 将上电后激光关闭的代码注释掉，发现反应速率提高，然而相同代码一段时间过后，无法正确解析数据，推测该传感器不好使！

### [2.2.0] - 2025-3-12

- 目前20Hz的传感器不能够满足拍球需求，后续需更换传感器或直接优化掉传感器
